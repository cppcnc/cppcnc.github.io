    1.0 Описание

    1.0.0 Станок НС-32.55.00 (далее «станок») предназначен для сверления ПП в
    автоматическом режиме.

    1.0.1. Для управления станком требуется компьютер. Компьютер приобретается
    отдельно. Минимальные требования к компьютеру см. в таблице.

    1.0.2. Исходной информацией для сверления является файл в формате «Exellon», на
    основе которого управляющее программное обеспечение станка(далее «ПО»)создает
    файл управления.

    1.0.3. Станок поставляется полностью подготовленный к работе и не требует
    предварительной наладки.

    1.0.4. Станок не требует специального постамента для установки и может
    быть установлен на любой стол выдерживающий вес 45 кг .

    1.0.5. Конструкция станка позволяет пользователю самостоятельно осущест-
    влять сервисное обслуживание и послегарантийный ремонт.
    Необходимые приспособления входят в комплект поставки.

    К оглавлению

    2.0 Подключение к компьютеру и установка ПО

    2.0.1. Подключение к компьютеру производится к любому свободному СОМ порту
    нуль-модемным кабелем(входит в комплект поставки).

    2.0.2 ПО управления станком входит в комплект поставки. ПО не требует ин-
    сталляции, его достаточно скопировать на жесткий диск.

    2.0.3 Предварительная настройка сводится к выбору COM порта через который
    будет осуществляться управление станком. После включения питания станка в строке
    уведомления ПО должна появиться строка «Устройство ОК».

    К оглавлению

    3.0 Подготовка к работе

    3.1 Создание файла Exellon

    3.1.0. Файл Exellon создается в программном пакете для проектирования ПП который Вы используете.

    3.1.1. Разместите проект максимально близко к началу координат. Именно от начала координат ПО станка
    будет отсчитывать перемещения инструмента.

    3.1.2. На настоящий момент ПО станка не обрабатывает все возможные настройки предусмотренные
    форматом Exellon. Для корректной обработки, при создании Exellon, необходимо задать следующие
    настройки:

    1) Задайте расширение - *.dri

    2) Output Units – milimeters
    3) Output Code Type – ASCII None
    4) Zero Suppression – None

    3.2 Подготовка управляющего файла

    3.2.0. Создание управляющего файла требует от пользователя выставления только одной настройки
    – создавать файл для прохода сверлами диаметров предусмотренных проектом или,
    проигнорировав эту информацию, создать управляющий файл под один диаметр сверла.

    3.2.1. Порядок создания управляющего файла:

    1) Запустите ПО станка.

    2) Выставьте переключатель одно/мульти инструментный режим
    по умолчанию соответствует последнему выбору

    3) Откройте требуемый файл Exellon
    (выставив фильтр расширений как *.dri)

    3.2.2. Все. На основе файла Exellon автоматически создан файл с тем же
    именем, но расширением *.cnc – он и является непосредственным управ-
    ляющим файлом для перемещения инструмента.

    3.2.3. Об особенностях сверления в одно/мульти инструментном режиме см.
    п. 4.1.4. и 4.1.5. .

    3.2.4. Если для проекта уже создавался управляющий файл, то достаточно
    его открыть(фильтр расширений по умолчанию как *.cnc).

    3.3 Подготовка и крепление заготовки ПП

    3.3.1. К заготовке предъявляется только одно требование – тщательно обработанные края для
    обеспечения плотного прилегания к подложке.

    3.3.2. Минимальные размеры заготовки 60х80 мм, максимальные 210х300мм,промежуточные
    величины определяются сеткой крепежных отверстий.

    3.3.3. Требуемый размер заготовки выбирается следующим образом:
    размер платы + минимум 5мм с каждой стороны для крепящих скоб.

    3.3.4. Заготовку необходимо крепить на подложку. Во первых для избегания
    контакта сверла с поверхностью рабочего стола, во вторых для уменьшения
    образующихся на выходе сверла заусенец. Размер подложки должен соответ-
    ствовать размеру заготовки. Желательно использовать специализированные материалы.
    При отсутствии такой возможности использовать любой подходящий материал
    толщиной 1.5 – 4.0 мм. Например лист ДВП, фанеры, алюминия(наилучший и самый дорогой вариант).

    3.3.5. Заготовка базируется в любом месте области сверления. Ориентация
    выбирается так, чтоб совпадало направление системы координат станка и системы CAD.

    3.3.6. Фиксация осуществляется крепежными скобами, притянутыми к рабочему столу зажимными винтами.

    К оглавлению

    4.0 Работа

    4.1 Сверление

    4.1.0. С помощью ручного управления подведите шпиндель к точке над заготовкой
    соответствующей НУЛЮ начала координат, визуально убедитесь, что траектория движения
    сверла не пересекается с крепежными скобами.
    4.1.1. Выставьте в ПО величину вертикального хода инструмента, соответствующую
    толщине заготовки.
    4.1.2. Нажимите кнопу «Сверлить».
    4.1.3. Если не включены шпиндель или пылесос появится соответствующее предупреждение.
    После включения достаточно повторно нажать кнопу «Сверлить».
    4.1.4. Если управляющий файл был создан для сверления одним диаметром сверла,
    то все отверстия будут насверлены за один проход. По окончании шпиндель будет автоматически
    перемещен в исходную точку.
    4.1.5. Если управляющий файл был создан для сверления сверлами, разных диаметров то по окончании
    прохода каждого диаметра, перемещение прекращается, шпиндель поднимается для смены сверла.
    После смены сверла необходимо нажать кнопку «Продолжить». После прохода последним диаметром шпиндель
    будет автоматически перемещен в исходную точку.

    4.2 Прерывание/приостановка/возобновление сверления

    4.2.0. На случай нештатных ситуаций предусмотрено ручное прерывание сверления.
    4.2.1. После нажатия кнопки «Сверлить» разблокируются кнопки «Прервать» и «Приостановить».
    4.2.2. По нажатии кнопки «Прервать» процесс сверления прекращается, шпиндель возвращается
    в начальную точку.
    4.2.3. По нажатии кнопки «Приостановить», кнопка «Приостановить» приобретает вид «Продолжить»,
    процесс сверления приостанавливается, продолжить сверловку можно нажатием кнопки «Продолжить».
    Если вместо кнопки «Продолжить» нажать кнопку «Прервать» процесс сверления прекращается,
    шпиндель возвращается в начальную точку.

    К оглавлению
